Recoger y lanzar un barril Oprime cuando estés cerca de un barril para recogerlo. Suelta para lanzarlo. Oprime hacia arriba y suelta para lanzarlo hacia arriba. Oprime hacia abajo y suelta para posarlo.
Formar un equipo Oprime cuando Diddy y Dixie estén juntos para recoger en brazos a tu compañero. Oprime para lanzarlo. Usa esta técnica para atacar a tus enemigos o para llegar a zonas aparentemente inaccesibles.
Montar / Desmontar animales amigos Oprime para montar sobre un animal amigo y oprime para desmontarlo. Podrás montar sobre Enguarde solo con acercarte a él.
Ataque voltereta / Ataque coleta Oprime para ejecutar un ataque voltereta con Diddy o un ataque coleta con Dixie. Ten en cuenta que algunos enemigos no se pueden vencer con este ataque. Durante un salto con Dixie, oprime y mantenlo oprimido para caer lentamente.
