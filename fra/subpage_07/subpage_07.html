Interrompre

Sur la Console Virtuelle, vous pouvez interrompre votre progression et continuer plus tard.
Tout d’abord, appuyez sur pour afficher le Menu HOME.
Dans le Menu HOME, sélectionnez le Menu Wii. Un point de sauvegarde sera créé et vous retournerez au Menu Wii.
Pour reprendre la partie à partir du point de sauvegarde, sélectionnez Démarrer sur l’écran de pré-visionnage de la chaîne.
  Lorsque vous reprenez une partie interrompue, le point de sauvegarde s’effacera. Vous ne pourrez plus reprendre votre partie à partir de ce point.
  Les données du point de sauvegarde ne peuvent être copiées sur une carte SD. Lorsque vous copiez des données sauvegardées sur une carte SD, le point de sauvegarde est effacé.
Si vous appuyez sur RESET ou que vous éteignez la console Wii ou la télécommande Wii pendant que vous créez un point de sauvegarde, vous perdrez votre point de sauvegarde. Si cela se produit, vous ne pourrez pas récupérer vos données. Soyez donc prudent.
